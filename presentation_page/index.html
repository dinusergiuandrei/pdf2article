AUTOMATIC MAGAZINES METADATA FILLING

Description

Automatic Magazines Metadata Filling is an application who uses algorithms of machine learning for metadata recognition like title, author and content in a pdf file.

How to use:

1. First of all you need to upload a pdf file from the "Browse" button or from the drag & drop option.


2. After the pdf file finishes to upload you have to press the "Send" button for the file to be analyzed.


3. After the pdf file is processed you will see marks on the title, author and content of the file.
